# Requirements

## Development Environment

To build and run this project, you need the following:

-   **Flutter SDK**: >=3.2.0 <4.0.0
-   **Dart SDK**: Compatible with Flutter version
-   **Platforms**: Android (SDK & Tools), iOS (Xcode & CocoaPods), Web (Chrome)

## Dependencies

The project relies on the following key packages:

-   `flutter_riverpod`: ^2.4.9 (State Management)
-   `go_router`: ^13.0.1 (Routing)
-   `firebase_core`: ^2.24.2 (Backend)
-   `firebase_auth`: ^4.16.0 (Authentication)
-   `cloud_firestore`: ^4.14.0 (Database)
-   `google_generative_ai`: ^0.2.0 (AI Features)

## Setup

1.  **Install Flutter**: Follow instructions at [flutter.dev](https://flutter.dev/docs/get-started/install).
2.  **Get Dependencies**: Run `flutter pub get`.
3.  **Run**: `flutter run`
