## ğŸ Getting Started

### Prerequisites

- Node.js (>= 18.0.0)
- npm or yarn
- A Firebase project

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/akibcse24/mydailytasktracker.git
   cd rt
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Environment Setup**:
   Create a `.env.local` file in the root directory and add your configuration (see `.env.example`).

4. **Run development server**:
   ```bash
   npm run dev
   ```

5. **Open the app**:
   Navigate to [http://localhost:3000](http://localhost:3000) to see the result.
